# SIBEC - Sistema Institucional de Gestión Becaria

## Universidad Tecnológica La Salle (ULSA)

Sistema web completo para gestionar y monitorear las horas sociales de estudiantes becados.

## 🎯 Características Principales

### Roles del Sistema

1. **Personal de Bienestar (Admin)**
   - Acceso total al sistema
   - Gestión de estudiantes y asignaciones
   - Validación y reportes globales
   - Configuración del sistema

2. **Jefatura de Carrera**
   - Supervisión de estudiantes por carrera
   - Aprobación de registros de horas
   - Reportes académicos

3. **Docente/Personal Responsable**
   - Registro de horas sociales
   - Gestión de estudiantes asignados
   - Historial de actividades

4. **Estudiante Becado**
   - Visualización de progreso personal
   - Historial de actividades
   - Consulta de asignación actual

## 📊 Módulos del Sistema

### 1. Dashboard
- KPIs personalizados por rol
- Gráficos de progreso
- Actividad reciente
- Notificaciones

### 2. Gestión de Estudiantes
- Registro de estudiantes becados
- Seguimiento de horas acumuladas
- Estado y progreso individual
- Filtros avanzados

### 3. Asignaciones
- Asignación de estudiantes a áreas
- Selección de responsables
- Gestión por cuatrimestre
- Visualización de áreas institucionales

### 4. Registro de Horas
- Formulario de registro rápido
- Cálculo automático de horas
- Descripción de actividades
- Historial de registros

### 5. Aprobaciones
- Revisión de registros pendientes
- Aprobación/rechazo con comentarios
- Estadísticas de aprobaciones
- Filtros por estado

### 6. Reportes
- Filtros personalizables
- Exportación PDF/Excel
- Estadísticas por carrera
- Resumen por área

### 7. Configuración
- Gestión de roles y permisos
- Áreas institucionales
- Cuatrimestres académicos
- Administración de usuarios
